Política de Privacidade – BancaFácil

Última atualização: 04 de agosto de 2025

A sua privacidade é importante para nós. Esta Política de Privacidade explica como o aplicativo BancaFácil, desenvolvido pela Haltt Games, coleta, usa e protege os dados dos usuários.

1. Coleta de Informações

O aplicativo não coleta informações pessoais sensíveis como nome, CPF, e-mail ou localização.

As únicas informações armazenadas são:

  • Valores inseridos manualmente pelo usuário (banca, apostas, resultados)
  • Preferências de tema e notificações

Esses dados ficam armazenados apenas localmente no dispositivo.

2. Uso das Informações

Os dados inseridos servem exclusivamente para gerar relatórios e acompanhar a evolução da banca.

No futuro, funcionalidades Premium poderão incluir backup em nuvem, com consentimento explícito.

3. Compartilhamento de Dados

BancaFácil não compartilha dados com terceiros.

Nenhuma informação é vendida ou transferida a outras empresas.

4. Permissões do Aplicativo

O app pode solicitar permissão para:

  • Enviar notificações diárias (opcional e controlado pelo usuário)

5. Segurança

Os dados são armazenados localmente com boas práticas de segurança.

Não há transmissão de dados pela internet, exceto em futuras atualizações com autorização do usuário.

6. Direitos do Usuário

  • Limpar dados dentro do app
  • Desinstalar o app para remover completamente os dados
